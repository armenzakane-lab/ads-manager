🛠️ Setup Guide

Complete these steps to connect your Google Ads account.

  • Go to Google Cloud Console → Create a project → Enable the Google Ads API
  • APIs & Services → Credentials → Create OAuth 2.0 Client ID (type: Web application). Add http://localhost:5000/auth/callback as an authorized redirect URI
  • Apply for a Google Ads Developer Token at ads.google.com → API Center (takes a few days for approval)
  • Enter your credentials below and click Save Settings, then connect your Google account
